## 🛠️ 구현할 기능 목록

### 1. 사용자 입력 처리
#### a. 경주할 자동차 이름
- 입력 메시지 출력
- 입력 유효성 확인 -> 쉼표(,)를 기준으로 5자 이하인지
- 이름 목록 저장
#### b. 시도할 횟수 (n)
- 입력 메시지 출력
- 입력 유효성 확인

### 2. 각 차수별 실행 결과 출력
1) 난수 생성
2) 생성된 난수를 전진 기준과 비교
3) 자동차 개수만큼 1~2 과정 반복
4) 자동차가 전진한 결과 출력
5) n만큼 1~4 과정 반복

### 3. 최종 우승자 출력
a. 단독 우승일 경우

b. 공동 우승일 경우 -> 쉼로 구분
